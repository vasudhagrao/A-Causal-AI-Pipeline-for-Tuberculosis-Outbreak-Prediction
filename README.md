# A-Causal-AI-Pipeline-for-Tuberculosis-Outbreak-Prediction
A five-stage causal AI pipeline for TB outbreak prediction, counterfactual policy estimation, and invariant transfer learning across 36 Indian states.
Overview
CausalTB is an end-to-end causal machine learning pipeline for tuberculosis outbreak prediction and public health policy evaluation applied to India's 36 states and union territories from 2020 to 2023. Unlike conventional predictive models that learn statistical correlations, this pipeline identifies the structural causal drivers of outbreak risk, models geographic disease spread through a state adjacency graph, quantifies the expected impact of policy interventions through counterfactual estimation, and ensures predictions generalise to unseen epidemic years through invariant transfer learning.
The pipeline is structured as 12 sequential Jupyter notebook sections, each building directly on the outputs of the preceding stage. All results, saved models, plots, and processed data files are generated programmatically and reproducible end to end.

Problem Statement
Existing TB surveillance models predict outbreak risk from historical data but cannot answer three questions that public health decision-makers need answered:

Why is a particular state at high outbreak risk?
What would happen if a specific policy intervention were implemented?
Will the model still work next year when epidemic conditions shift?

CausalTB addresses all three questions within a single integrated pipeline.
